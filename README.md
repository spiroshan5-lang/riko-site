# RIKO Website

The RIKO hospitality website, migrated to Next.js App Router, TypeScript, and Tailwind CSS.

## Scripts

- `npm run dev` — start the local development server
- `npm run build` — generate an optimized production build
- `npm run start` — serve the production build
- `npm run lint` — run Next.js lint rules

## Structure

- `src/app` — Next.js routes and page metadata
- `src/components` — layout, section, and motion components
- `src/assets` — original website imagery, carried forward through Next.js static image imports
- `src/app/globals.css` — the original Tailwind theme, utility classes, and animation definitions
